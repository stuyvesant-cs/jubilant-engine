At any point, you can reassign (using =) a variable to any value you like, regardless of type. When you do this, it will update the type of the variable automatically. So the following code is valid: 
s = "this is a string!"
print(s)
s = -84
print(s - 3)
Just to be clear, if we didn't have s = -84, then the subsequent line would not work, resulting in a TypeError.
