With one spectial exception, if you use 2 values of the same number type (integer or floating point), the result will be a value of the same type. If the types are mixed, the result will be a floating point. The one exception is /, which always returns a floating point value, regardless of whether the divsion could result in an integer value. 
print(10 * 5)
print(8.4 + 2.3)
print(10 - 3.0)
print(32 / 8)
print(32 / 5)
