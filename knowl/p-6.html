While simple, the code above does demonstrate the most useful function in Python, print(). At its core, print() will attempt to display whatver arguments you pass into it in the console, followed by a newline. What's quite nice about print() is that you can give it multiple arguments, separated by , and it will print each agrument to the console on one line, separated by spaces. So: 
print("Hi", 5)
Will output:
        Hi 5
